# Installation

- Cloner le projet depuis l'addresse : [_ASETML/P_GestProj426_](https://github.com/ASETML/P_GestProj426 "Projet sur Github")
- Dans la racine du projet à : `\P_GestProj426`, ouvrir une console .bash Puis faires les commandes suivantes:  
  `cd Docker/`  
  `docker compose up -d`  
  `cd ../a`  
  `cd ../a`  
  `cd ../application/`  
  `npm install`  
  `npm start`

- Copier le `config.mjs.example`, le renommer en config.mjs. puis remplir les information nécessaires dans le fichier config.mjs  
  `const db_name = "";`  
  `const db_user = "";`  
  `const db_password = "";`  
  `const db_host = "";`  
  `const app_port = ;`  
  `const db_port = ;`
- Le serveur et siteweb sera maintenat 100% opérationnel.
